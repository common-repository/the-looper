== Description ==
Intuitive and friendly Admin UI that will create, edit and delete custom post types as well as custom taxonomies for WordPress. You can quickly register and manage custom post types and taxonomies settings making your website process more efficient and time sensitive.

== Installation ==
= ADMIN INSTALL VIA SEARCH =
1. Visit the Add New plugin screen and search for \"the looper\".
2. Click the \"Install Now\" button.
3. Activate the plugin.
4. Navigate to the \"The Looper\" Menu.

= ADMIN INSTALL VIA ZIP =
1. Upload `the-looper.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
